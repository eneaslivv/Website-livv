1/ Most companies that try to integrate AI pick the wrong workflow first and wonder why it doesn't pay back. Here's the framework that actually works. ↓

2/ Start with a workflow inventory. Map your 20 most significant workflows. Estimate annual staff hours and error rate for each. The top 3-4 by volume × time cost × room for improvement are your candidates. Start with the one that has the clearest scope — not the easiest one.

3/ Three integration patterns, three cost tiers:
- Buy an AI-native tool: $50–$400/user/month
- API integration into an existing app: $8k–$25k dev + $200–$2k/mo API usage
- Custom AI agent: $30k–$120k depending on tool complexity

4/ The failure mode nobody talks about: the PoC that never ships. Successful demo, compelling results, six months of "review." Fix: define a success criterion before you run the PoC. "If this cuts review time by 30%, we ship within 60 days." Ship-or-kill, not open-ended.

5/ Prompt engineering by committee kills quality. When 5 stakeholders each add opinions to a system prompt, the output degrades. One owner. An evaluation process. Iteration. Not consensus management.

6/ Budget anchors for 2026:
- Single AI feature: $8k–$40k
- RAG knowledge system: $20k–$80k
- Custom agent: $30k–$120k
- Maintenance: 15–25% of build cost per year
- PoC cap: 4 weeks, defined budget, defined kill criterion
